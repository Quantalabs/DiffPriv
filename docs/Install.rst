Installing
##########

Installing from the pip
=======================

Use:

.. code-block::

   pip install DiffPriv

or:

.. code-block::

    pip install DiffPriv==0.1.1

to install.

Using DiffPriv in your programs
===============================

When using it in your program, use:

.. code-block:: python

    import DiffPriv as dp

Now you're ready to start using DiffPriv!
